# My First Backend

Projeto full-stack com Node.js + Express + React + TypeScript.

## Estrutura do Projeto

```
my-first-backend/
├── client/          # Frontend React com Vite
│   ├── src/
│   ├── public/
│   └── package.json
│
├── server/          # Backend Node.js com Express
│   ├── src/
│   ├── config/
│   ├── controllers/
│   ├── models/
│   ├── routes/
│   └── package.json
│
└── README.md
```

## Como Executar

### Backend (Servidor)

```bash
cd server
npm install
npm run dev
```

### Frontend (Cliente)

```bash
cd client
npm install
npm run dev
```

## Tecnologias

### Backend

- Node.js
- Express
- TypeScript
- MySQL
- JWT
- bcrypt

### Frontend

- React 19
- TypeScript
- Vite
- Tailwind CSS
- React Router
